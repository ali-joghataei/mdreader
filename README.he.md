# MdReader

[English](README.md) | [فارسی](README.fa.md) | [العربية](README.ar.md) | עברית | [اردو](README.ur.md)

MdReader היא אפליקציית שולחן עבודה חוצת-פלטפורמות לפתיחה, קריאה ועריכה של קבצי Markdown.

המיקוד המרכזי שלה הוא הצגה נקייה של Markdown בעברית ובשפות אחרות הנכתבות מימין לשמאל. היא מיועדת להתמודד טוב יותר עם תוכן מעורב RTL/LTR מאשר שימוש פשוט ב-`dir="auto"`, במיוחד כאשר פסקה מתחילה במילים באנגלית, שמות מחלקות, שמות בדיקות או קוד inline.

הפרויקט הזה נכתב כולו בעזרת בינה מלאכותית.

## יכולות

- פתיחת קבצי Markdown מתפריט File
- פתיחת קבצים באמצעות גרירה ושחרור
- פתיחת כל קובץ בחלון נפרד
- מצב קריאה ותצוגה מקדימה כברירת מחדל
- מצב עריכה אופציונלי עם תצוגה מקדימה חיה
- Split view לעריכה ותצוגה מקדימה בו-זמנית
- סימון לשינויים שלא נשמרו
- בחירת גופן מתוך הגופנים המותקנים במערכת
- שמירת הגדרות האפליקציה
- הצגה טובה יותר של Markdown מימין לשמאל ותוכן דו-כיווני
- Toggle Developer Tools מתפריט האפליקציה
- תמיכה בשיוך קבצי Markdown בגרסאות ארוזות

## תמיכה מימין לשמאל

MdReader מזהה את כיוון הטקסט של בלוקי Markdown בצורה חכמה יותר מאשר `dir="auto"` פשוט.

זה חשוב עבור Markdown טכני בעברית, בערבית, בפרסית, באורדו ובשפות RTL נוספות, מפני שטקסטים כאלה כוללים לעיתים קרובות מזהים באנגלית, קוד inline, שמות בדיקות, שמות enum או מונחים של frameworks.

האפליקציה מנתחת כל בלוק מוצג ומחילה כיוון RTL כאשר רוב התוכן הוא מימין לשמאל, תוך שמירה על קריאות של קוד וחלקים טבעיים משמאל לימין.

## פיתוח

```powershell
npm install
npm run dev
```

## Build

בניית האפליקציה:

```powershell
npm run build
```

בניית installer ל-Windows:

```powershell
npm run dist:win
```

או שימוש בסקריפט העזר:

```powershell
powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\scripts\build-installer.ps1
```
