# MdReader

[English](README.md) | [فارسی](README.fa.md) | العربية | [עברית](README.he.md) | [اردو](README.ur.md)

MdReader هو تطبيق سطح مكتب متعدد المنصات لفتح ملفات Markdown وقراءتها وتحريرها.

يركز التطبيق على عرض Markdown بشكل واضح للغة العربية واللغات الأخرى التي تُكتب من اليمين إلى اليسار. صُمم للتعامل مع المحتوى المختلط RTL/LTR بطريقة أفضل من استخدام `dir="auto"` فقط، خاصة عندما تبدأ الفقرة بكلمات إنجليزية أو أسماء أصناف أو أسماء اختبارات أو كود inline.

تمت كتابة هذا المشروع بالكامل بمساعدة الذكاء الاصطناعي.

## الميزات

- فتح ملفات Markdown من قائمة File
- فتح الملفات بالسحب والإفلات
- فتح كل ملف في نافذة مستقلة
- وضع القراءة والمعاينة بشكل افتراضي
- وضع تحرير اختياري مع معاينة مباشرة
- Split view للتحرير والمعاينة في الوقت نفسه
- مؤشر للتغييرات غير المحفوظة
- اختيار الخط من الخطوط المثبتة على النظام
- حفظ إعدادات التطبيق
- عرض أفضل لمحتوى Markdown من اليمين إلى اليسار والمحتوى ثنائي الاتجاه
- Toggle Developer Tools من قائمة التطبيق
- دعم ربط ملفات Markdown في النسخ المعبأة

## دعم اليمين إلى اليسار

يحدد MdReader اتجاه كتل Markdown المعروضة بطريقة أذكى من `dir="auto"` البسيط.

هذا مهم لملفات Markdown التقنية المكتوبة بالعربية أو الفارسية أو العبرية أو الأردية أو أي لغة RTL أخرى، لأن هذه النصوص غالبا تحتوي على معرفات إنجليزية أو كود inline أو أسماء اختبارات أو أسماء enum أو مصطلحات خاصة بالأطر البرمجية.

يفحص التطبيق محتوى كل كتلة معروضة ويطبق اتجاه RTL عندما يكون المحتوى غالبا من اليمين إلى اليسار، مع إبقاء الكود والأجزاء الطبيعية LTR قابلة للقراءة.

npm run build
