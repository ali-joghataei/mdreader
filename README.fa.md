# MdReader

[English](README.md) | فارسی | [العربية](README.ar.md) | [עברית](README.he.md) | [اردو](README.ur.md)

MdReader یک اپلیکیشن دسکتاپ کراس‌پلتفرم برای باز کردن، خواندن و ویرایش فایل‌های Markdown است.

تمرکز اصلی آن نمایش تمیز Markdown فارسی و دیگر زبان‌های راست‌به‌چپ است. برنامه برای محتوای ترکیبی RTL/LTR بهتر از یک `dir="auto"` ساده عمل می‌کند؛ مخصوصا وقتی پاراگراف با کلمه انگلیسی، نام کلاس، نام تست یا کد inline شروع شده باشد.

این پروژه به‌طور کامل با کمک هوش مصنوعی نوشته شده است.

## ویژگی‌ها

- باز کردن فایل‌های Markdown از منوی File
- باز کردن فایل با drag and drop
- باز شدن هر فایل در یک پنجره جداگانه
- حالت پیش‌فرض خواندن و پیش‌نمایش
- حالت اختیاری ویرایش با پیش‌نمایش زنده
- Split view برای ویرایش و پیش‌نمایش همزمان
- نمایش وضعیت ذخیره‌نشده برای فایل‌های تغییرکرده
- انتخاب فونت از فونت‌های نصب‌شده سیستم
- ذخیره تنظیمات برنامه
- نمایش بهتر Markdown راست‌به‌چپ و دوزبانه
- Toggle Developer Tools از منوی برنامه
- پشتیبانی از file association برای Markdown در نسخه بسته‌بندی‌شده

## پشتیبانی راست‌به‌چپ

MdReader جهت بلاک‌های رندرشده Markdown را هوشمندتر از `dir="auto"` ساده تشخیص می‌دهد.

این موضوع برای Markdown فنی فارسی، عربی، عبری، اردو و دیگر زبان‌های راست‌به‌چپ مهم است؛ چون چنین متن‌هایی معمولا شامل شناسه‌های انگلیسی، کد inline، نام تست، نام enum یا اصطلاحات فریم‌ورکی هستند.

برنامه محتوای هر بلاک رندرشده را بررسی می‌کند و وقتی متن غالبا راست‌به‌چپ باشد، جهت RTL اعمال می‌کند؛ در عین حال کدها و بخش‌های طبیعی چپ‌به‌راست خوانا باقی می‌مانند.

## اجرای توسعه

```powershell
npm install
npm run dev
```

## ساخت

Build برنامه:

```powershell
npm run build
```

ساخت installer ویندوز:

```powershell
npm run dist:win
```

یا استفاده از اسکریپت آماده:

```powershell
powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\scripts\build-installer.ps1
```
